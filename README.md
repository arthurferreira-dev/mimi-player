# Mimi Player

Mimi Player é um player de música inspirado no Reprodutor Multímida do Windows e no Lark Player. É leve, rápido e útil!

## Atalhos

* `Space`: Pause/Play a música
* `R`: Ativa/Desativa o Replay
* `F`: Muta/Desmuta a música

## Colocando a música

Você pode clicar no botão escolher música e selecionar um arquivo de aúdio que preferir!
